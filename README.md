# 👁️ NeuralVision | AI/ML Intelligence Dashboard

A high-fidelity dashboard mock designed to visualize computer vision processing, model accuracy, and real-time inference statistics.

## ✨ Features
- **Vision Console**: Mock live feed with scanning line animations.
- **Model Analytics**: Visual representation of Accuracy, Inference Time, and Model Health.
- **Modern Dashboard UI**: Sidebar navigation with high-performance metrics cards.
- **Glassmorphism Design**: Sleek black and blue aesthetic inspired by modern AI tools.

## 🛠️ Tech Stack
- **Languages**: HTML, CSS, JS.
- **Concepts**: Computer Vision, Deep Learning Visualization, Data Analytics.

## 🧠 Why I built this?
Artificial Intelligence is the future of CSE. I created this dashboard to demonstrate how complex ML model outputs can be presented clearly to end-users via professional-grade interfaces.
